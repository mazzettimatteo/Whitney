<h1>Whitney colorscheme for Vim/Neovim</h1>
<p>A Vim colorscheme for multiple languages (such as C, C++, Python, Java, Javascript, etc.) created by Matteo Mazzetti.</p>
<h2>Colors</h2>
<ul>
  <li>Black background;</li>
  <li>Soft green conditionals and loops;</li>
  <li>Purple types;</li>
  <li>Number literals in Almond Silk (#FFCDBC);</li>
</ul>
<h2>Installation</h2>
<h3>Using Vim 8+ or Neovim</h3>
<p>Copy the <b>colors/Whitney.vim</b> file into your <b>~/.vim/colors/</b> (or <b>~/.config/nvim/colors/</b> for Neovim).</p>
<h2>Using the colorscheme</h2>
<p>In Vim use <b>:syntax on</b> and <b>:colorscheme Whitney</b> or, if you want to set it as default, add <b>colorscheme Whitney</b> in the Vim config file <b>~/.vimrc</b>.</p>
